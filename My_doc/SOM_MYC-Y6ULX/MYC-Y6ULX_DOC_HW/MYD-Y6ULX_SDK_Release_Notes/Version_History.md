---

## ⚠️ Warm Tips

1. MYIR does **not** supply maintenance service for LCD. Check the LCD within **7 business days** of receiving goods; contact MYIR immediately if there is no display.
2. Do **not** use fingernails or hard/sharp objects on the LCD surface.
3. Use a dedicated screen wiper to clean the LCD after long-term use; avoid using fingers/hands.
4. Do **not** clean the screen surface with chemicals.
5. Read the product user manual thoroughly before use.
6. For any maintenance, communicate with MYIR first to confirm the issue. MYIR will issue an **RMA number** for return service after confirmation.
