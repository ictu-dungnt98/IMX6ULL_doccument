# Chapter 1: SDK Overview

The design and development of MYD-Y6ULX SDK software is based on the **ARM® Cortex®-A7** core, which includes:

- Underlying BSP source code
- Pre-compiled image files
- Linux software evaluation and development documents
- Tools used in the development and debugging process

Corresponding hardware information is also released along with the SDK in the form of a CD image.
